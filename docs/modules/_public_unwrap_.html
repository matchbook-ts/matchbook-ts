    • Defined in src/public/unwrap.ts:38
    description

    This throws if the match was not exhaustive, and returns R if it matched.

    I sure hope you know what you are doing!

    throws

    UnwrapError

    since

    1.0.0

    example
    enum Time {
    Morning = 'Morning',
    Noon = 'Noon',
    Afternoon = 'Afternoon',
};
const getMessage = pattern(
    match(Time.Morning, 'Good morning!'),
    unwrap(),
);
try {
    // This will throw!!
    getMessage('blarg!');
} catch (e) {
    if (e instanceof UnwrapError) {
        console.error(e.message);
    }
}
